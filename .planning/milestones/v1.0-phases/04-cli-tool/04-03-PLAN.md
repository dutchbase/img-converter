  <behavior>
detectFormatFromExt:
- ".jpg" → "jpeg", ".jpeg" → "jpeg"
- ".png" → "png", ".webp" → "webp", ".avif" → "avif"
- ".gif" → "gif", ".tiff" → "tiff", ".tif" → "tiff"
- ".heic" → "heic", ".heif" → "heic"
- ".xyz" → null (unknown)
- ".JPG" → "jpeg" (case-insensitive via toLowerCase)

buildOutputPath:
- ("/photos/x.jpg", "webp") → "/photos/x.webp" (same dir, new ext)
- ("/photos/x.jpg", "webp", "/out") → "/out/x.webp" (explicit outputDir)
- File with no extension: basename = full filename, ext = ""

buildConvertOptions:
- Maps opts.format → targetFormat (cast to ImageFormat)
- Maps opts.quality → quality (number, default 85)
- opts.width → resizeWidth (number | null, null if undefined)
- opts.height → resizeHeight (number | null, null if undefined)
- maintainAspectRatio: true (fixed — CLI does not expose this flag)
- opts.metadata === false → removeMetadata: true (Commander negation inversion — CRITICAL)
- opts.metadata === true → removeMetadata: false
- allowUpscaling: undefined (not exposed in v1)

formatKB:
- 0 → "0 KB"
- 1024 → "1 KB"
- 433152 → "423 KB"
- Rounds to nearest integer

isPipeMode:
- (isTTY=false, files=[]) → true
- (isTTY=false, files=["a.jpg"]) → false
- (isTTY=true, files=[]) → false
  </behavior>
